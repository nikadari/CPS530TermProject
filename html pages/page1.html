Front-end: React Popularity,Strength and Weaknesses

React is really popular for building for web and mobile apps. React's strengths are it can improve UX and coding experience using Virtual Document Object Models (DOM). You can reuse the code elements you already created. It can produce substantial code using the single-oriented data stream. It has losts of open-source code libraries. React's weaknesses are extreme development velocity since there are new updates and the user had to learn them all. Since it updates so fast there is insufficient documentation for having manual for React.

Back-end: Django Popularity,Strength and Weaknesses

Django is really popular with any type of websites. Django prides itself as a batteries-included network. You can just import packages that you want to use. It is scalable network. It has a build-in admin. Since Django uses Pyhton it is easy to learn and to use. Django weaknesses are URL specifying with regular expressions is not an easy task to accomplish, at least for beginners. It also feels a little bloated for small projects, and some people find it quite populated with big projects since models, for example, are all included in a single file. Django is too monolithic.
