How to Create a Good Looking Page in React

Step 1

Split page elements into components.
Each component requires a .js file and .css file (with same name).

Step 2

In the .js file, you will write HTML code to establish the component's layout and items. This file will be used to hold text, images, icons and other HTML elements.

Step 3

In the .css file, you will write CSS code to stylize the HTML elements and attributes from previous step.

Step 4

Add Text (using span, h1-h6, p) and style the attributes by customizing HTML text tags using the .css file from previous step. (Import stylesheet at the top of .js file!)

Step 5

Add Images (using img, figure) and adjust their position by wrapping images inside div and styling using margin and padding style attributes.

Step 6

Once finished with the component .js file, open the built in App.js file in the React application src folder. Import the component at the top of App.js. Use your component name as a React element (similar to an HTML tag) to display onto the page.
